Fonctionnement :

maitre du jeu lance un combat.
boucle :
    MJ envoie un combat à la page via controlleur
    Page envoie choix du joueur au MJ via controlleur.


Mise en oeuvre :
    Page dessinerCombat :
        supprime l'ensemble du plateau.
        ajoute le dessin du joueur actif
        2 boutons : attaque normal, attaque kamikaze
            on click, envoie choix au maitre du jeu.
        if message:
          if (succès/échec) : ecrire message.
          else: dessinerFinPartie

    Page dessinerFinPartie:
        Supprime tout les éléments
        Ajoute un dessin GAME OVER
        Ajoute un bouton reload (actualise la page)

    MJ lancer combat:
        demander à la page de dessiner le combat

    MJ gerer combat:
        recoit choix du joueur depuis la page.
        met en oeuvre le choix.
        change le joueur actif.
        if joueurMort :
            renvoi (FIN)
        else : renvoit un message (succès/échec attaque)
